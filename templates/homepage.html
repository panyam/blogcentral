Usage

Installation

BlogCentral is available on the G Suite Marketplace. Select Install to begin using it in Google Docs.

Starting the add-on

Open a new doc and Click on Add-Ons >> BlogCentral >> Start.

Contributing

Setup

            
            $ cd gdocs
            $ npm install

Build

            
            $ cd gdocs
            $ npm run build

Pushing Changes

Command Description
npm run push:server All GAS Server Scripts
npm run push:client just main.html
